## 1. 백준 9021 괄호 [>> 링크](https://www.acmicpc.net/problem/9021)


```python
T = int(input())

stack = []

for _ in range(T):
    stream = list(input().rstrip())

    is_VPS = True

    for i in range(len(stream)):
        if stream[i] == '(':
            stack.append(stream[i])
        else:
            if stack:
                stack.pop()
            else:
                is_VPS = False
                break
    
    if not stack:
        if not is_VPS:
            print("NO")       
        else:
            print("YES")
    else:
        print("NO")
    

    stack.clear()
```
